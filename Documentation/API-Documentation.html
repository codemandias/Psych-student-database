createAdmissionRecord createAdmissionRecord (optional)

Return type

AdmissionRecord

Content-Type: application/json
{

  "admission_date" : "01-01-2020",

get /get-admission-record/
returns admission records for a student (getAdmissionRecord)
By passing in student banner number gets admission records for that student

Query parameters

student_id (required)
Query Parameter — pass in the banner number/student id to retrieve that student's admission record

Return type

AdmissionRecord

Example data

Content-Type: application/json
{
  "admission_gpa" : 4.0,
  "admission_date" : "01-01-2020",
  "previous_school" : "St. Mary's University",
  "AdmissionNotes" : "additional notes",
  "is_foreign" : "0 or 1",
  "student_id" : "B00123123",
  "previous_degree" : "Bachelors of Science",
  "admission_id" : ""
}

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Student record found AdmissionRecord

400

Invalid ID supplied

404

Student Record not found
